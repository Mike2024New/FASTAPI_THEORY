APP1 - приложение сократитель ссылок, практика по изученным темам. Приложение учебное, БД упрощена до json для хранения
url. Нужно будет в json менеджере обеспечить асинхронность а то при большом количестве запросов возможно состояние гонки
и повреждении данных в json, в остальном все работает ок. Будет интересно сделать ещё ui, и в целом это не последняя
версия.

APP2 - не претендует на урок, просто учебный пример эксперимент для закрепления практических навыков по теме работы с
ссесиями обработки данных из форм. А также рассмотрено как сделать простейшую регистрацию (с сильными упрощениями).
