Раздел уроков по фронтенду fastapi приложений

Шаблонизатор JINJA2 и html страницы в fastapi:
    [Шаблоны jinja2 хороши, тем что могут помочь сверстать ui для своих приложений, можно не только сделать энпоинты
    работающие на сухом json, но и простой и даже средней сложности приложения, когда не требуется выносить логику
    работы frontend части в отдельный слой]

    lsn10_1 - jinja2 простая подстановка значений вместо переменных -> {{ variable }}.
    отправка пользователю html страницы с заполнением страницы данными с сервера. Использование
    Jinja2Templates для того, чтобы указать расположение директории с шаблонами передать в шаблон данные и отправить
    пользователю. Подстановка в шаблон данных, в шаблоне такие данные обозначаются {{ variable }}, и с сервера
    необходимо передать переменную которая содержит значение этой переменной. Передача выполняется через контекст.

    lsn10_2 - циклы в jinja2 {% for var in array %}, в шаблонах можно использовать циклы, но их также нужно закрывать с
    помощью {% endfor %}. Внутри цикла доступен объект loop, который содержит в себе полезную информацию о состоянии
    цикла, например loop.index - номер текущей итерации (начиная с 1), что может быть очень удобным для нумерации строк.
    За пределами цикла loop не доступен, и если нужно узнать длину коллекции, то используется опция length, например
    {{ arr | length }}.
